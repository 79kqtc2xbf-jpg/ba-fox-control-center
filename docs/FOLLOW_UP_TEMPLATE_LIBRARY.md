# BA Fox — Follow-up Template Library

## Purpose

This document stores approved follow-up templates for BA Fox / Executive Fox.

The templates support Stage 4 hybrid Gmail reconciliation:

```text
ChatGPT checks Gmail → suggests status → prepares follow-up → Lisa confirms → Lisa sends / updates sheet
```

The templates should be:

- short;
- professional;
- ready to copy;
- adaptable;
- not too formal unless needed;
- clear about next step;
- written without invented facts.

## Usage rule

Never auto-send these messages in MVP.

Use them as drafts for Lisa to review, copy, edit and send.

## Variables

Use placeholders:

```text
[Name]
[Company]
[Topic]
[Date]
[Document package]
[Next step]
[Meeting date/time]
[Lisa signature]
```

Default signature:

```text
Best regards,
Lisa
Business Assistant
MF Group
```

or for MF Corp:

```text
Best regards,
Lisa
MF Corp CO., LTD
```
