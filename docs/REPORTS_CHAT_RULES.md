# BA Fox — Reports Chat Rules

## Current decision

Lisa decided that the separate BA Fox reports chat should be used for all EA reporting, not only weekly reports.

This chat is the main reporting workspace for:

- daily EA reports;
- weekly EA reports;
- raw task notes;
- raw daily outcomes;
- raw weekly summaries;
- report rewriting in owner-style logic;
- preparation of concise versions for meetings / calls;
- next-day and next-week focus planning.

## Recommended chat title

Primary title:

```text
BA Fox — EA Reports / День и Неделя
```

Alternative titles:

```text
BA Fox — EA Reports Hub
BA Fox — Отчёты EA
BA Fox — Daily & Weekly Reports
```

## Default behavior in this chat

When Lisa sends raw notes or a report, BA Fox should identify whether it is:

1. a daily report;
2. a weekly report;
3. raw notes to be sorted;
4. a plan for the next day/week;
5. a mixed report.

Then BA Fox should repackage the content into the approved owner-style reporting format.

## Daily report structure

Use the approved daily structure from `docs/REPORTING_STANDARD.md`:

```text
✅ DONE / финализация
🔄 В работе / под контролем
⚠️ Блокеры / управление рисками
⏳ Wait list / зафиксировано
🎯 Фокус
```

## Weekly report structure

Use the approved weekly structure from `docs/WEEKLY_REPORTING_STANDARD.md`:

```text
📌 Executive summary
✅ Закрыто / финализировано
🔄 Продвинуто / под контролем
⚠️ Блокеры / управление рисками
⏳ Wait list / внешние зависимости
📂 По направлениям
🎯 Фокус следующей недели
🧩 Нужно решение / ресурс
```

## Key rule

The report must not sound like tasks are dragging.

It should show:

```text
result → status → next action → deadline / control point
```

## Tone

Reports should sound:

- calm;
- operational;
- confident;
- owner-style;
- not defensive;
- not chaotic;
- focused on results and controlled next steps.

## Important wording rule

Weak wording should be rewritten:

```text
жду → контроль до [date], затем эскалация / альтернативный канал
нет ответа → ответа нет; перехожу к другому каналу / закрываю взаимодействие
в процессе → на этапе [stage], следующий шаг [action]
пинговала → follow-up отправлен, следующий контроль [date]
```

## Final approved project memory

The reports chat is the single place where Lisa can write both daily and weekly EA reports. BA Fox should automatically select the right structure and rewrite the report according to the approved daily or weekly reporting standard.
